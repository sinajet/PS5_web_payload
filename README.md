# PS5 Web Payload

A webkit-based Payload sender for PS5.

## Description

This host is for sending payloads on elfldr by john-tornblom, not for jailbreaking (Only works on PS5.

## Usage Guide

1. Download the browser shortcut host package from the Releases section.

2. Load the etaHen or kstuf payload.

3. Go to the Debug Settings on the console and install it.

4. Ensure the console is properly connected to the internet.

5. Navigate to the Media section on the PS5 console.

6. Open the "PS5 web Payload" app and run it.

## Compatibility

This host works up to firmware 5.50. Higher firmwares have not been tested.
